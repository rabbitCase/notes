- Create a database
 `**CREATE DATABASE cm_devices;**`
 - Use/select that database to create tables in it
 `**USE cm_devices;**`
 - Create a table
 `**CREATE TABLE devices (deviceID int, deviceName varchar(50), price decimal);**`
 - View the table
 `SHOW tables`
 - View specific columns
 `**SHOW columns FROM devices;**`
